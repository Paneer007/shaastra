## Getting Started

- Create a meta mask account
- Install the dependencies using 
```bash
npm install
```

- To run the server along with the ComposeDB, run
 ```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack used
- ComposeDB
- Ceramic Network
- GraphQL
- Next.js
- Typescript
- Sass
